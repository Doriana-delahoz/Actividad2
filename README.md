# Actividad2

## Sobreescritura de metodos
 es cuando una clase hija reemplaza un metodo de la clase padre para realizar una accion diferente.

 ## Parametros nombrados
 Son aquellos que se envian indicando el nombre de cada parametro el momento de llamar una funcion o metodo

 ## Parametros posicionales 
 Son parametros que deben enviarse en el mismo orden en que fueron declarados
